Всё идёт
к агентам.

Два года назад нейросети только подсказывали — что нажать, как написать. Сегодня они делают это сами. Открывают файлы, заходят в сервисы, доводят задачу до конца.

Ты говоришь словами — агент делает руками. Ставит подзадачи, открывает файлы, ходит в сервисы, проверяет себя и приносит готовое.

Таких инструментов уже много. Нам нужны три.

Агенты, оркестраторы, мультиагентные команды, MCP-тулзы, своя сборка на API — большая часть этого шум. Запомни три названия. Остальное — потом, если упрёшься.

1
OpenAI ChatGPT → Codex
Codex
Хорош в коде и системных задачах. Подходит, если уже сидишь на ChatGPT Plus.
2

Claude Code
Самый универсальный. Тащит сложные проекты, на нём сидят специалисты.
3
Google Gemini → Antigravity
Antigravity
Сильный для дизайна и базовых задач. Менее универсален, чем Claude Code.

Все три — рабочие. У каждого свои сильные стороны. Но если выбирать одного — выбирай лучшего.

Бери Claude Code.
⚠Общий минус всех трёх. ChatGPT, Claude и Gemini могут заблокировать аккаунт без объяснений. Иногда за дело, иногда нет. На «ВАЙБе» показываем, как этого избежать.

Где его теперь
открыть?

Claude Code запускается по-разному. Часть способов — чтобы потыкать. Один — чтобы реально работать.

У него есть сайт (claude.ai/code), мобильное приложение и программа для компьютера. Каждый вариант — под свою задачу.

Каждый, кто сидит на своём способе, скажет, что прав именно он. Разберём по порядку.

Сайт, приложение и телефон — это чат с моделью: задал вопрос, получил ответ. К твоим файлам они не подключаются — реальный проект через них не сделать.

Терминал — чёрное окно с командами на компьютере. Claude Code там уже видит и правит твои файлы, но показывает всё текстом, без интерфейса.

Cursor — окно, где сразу видно файлы проекта, их содержимое и что с ними делает агент в реальном времени. На нём и работают.

Важно: Cursor не заменяет Claude Code. Cursor — это окно с твоими файлами. Claude Code — агент, который в этом окне с файлами работает. Без Cursor его некуда поставить, без Claude Code — нечем работать.

На «ВАЙБе» ставим Cursor и Claude Code на твой компьютер в первый вечер. На твоём проекте, не учебно.
