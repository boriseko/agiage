## TIER S - MUST STUDY (топ-10, обязательно к изучению)

### 1. Stripe.com
- **URL**: https://stripe.com
- **Что крутого**: Эталон mesh-gradient анимации (синий-фиолетовый-розовый), идеальный whitespace, sharp типографика. Каждый элемент в движении, но не раздражает. Gradient effect имеет open-source туториалы.
- **Можно стырить**: Нет (custom build), НО gradient effect документирован: https://kevinhufnagl.com/how-to-stripe-website-gradient-effect/
- **Платформа**: Custom React

### 2. Linear.app
- **URL**: https://linear.app
- **Что крутого**: Породил целый тренд "linear design" — sequential, scannable layout, bold типографика, glassmorphism, shimmer эффекты. Smooth transitions. В 2025 перешли на монохром.
- **Можно стырить**: ДА! Open-source клон: https://github.com/frontendfyi/rebuilding-linear.app (Next.js + Tailwind + Framer Motion)
- **Платформа**: Custom React/Next.js

### 3. Notion.com
- **URL**: https://www.notion.com
- **Что крутого**: Тёплые пастельные цвета, playful кастомные иллюстрации, моментально узнаваемый бренд. Лёгкая тема, дружелюбная и approachable. Bento grid для фич.
- **Можно стырить**: Нет (custom), но Notion Sites позволяет строить похожие страницы
- **Платформа**: Custom React

### 4. Amie.so
- **URL**: https://www.amie.so
- **Что крутого**: Золотой стандарт motion design на светлом фоне. Framer Motion анимации, UseInView hook, layout animations. Стала референсом для всего design community.
- **Можно стырить**: Анимации можно воспроизвести: https://www.frontend.fyi/tutorials/recreating-amie-so-animations-with-framer-motion
- **Платформа**: Custom (Framer Motion + Tailwind)

### 5. Vercel.com
- **URL**: https://vercel.com
- **Что крутого**: Чистый чёрно-белый с стратегическими gradient акцентами. Geist typeface, grid layout, smooth page transitions. Benchmark для developer-focused дизайна.
- **Можно стырить**: Geist шрифт и компоненты open-source. Web Interface Guidelines публичны: https://vercel.com/design/guidelines
- **Платформа**: Next.js + Tailwind

### 6. AuthKit by WorkOS
- **URL**: https://www.authkit.com
- **Godly**: https://godly.website/website/authkit-991
- **Что крутого**: Crisp белая основа с azure blue, soft purple акценты, yellow highlights. Bold sans-serif, illustrative line art, micro-animations. React + Next.js + Untitled Sans.
- **Можно стырить**: Нет, но layout pattern легко реплицируется
- **Платформа**: Custom (React, Next.js)

### 7. Anthropic.com
- **URL**: https://anthropic.com
- **Что крутого**: Чистый, минимальный подход к коммуникации сложных AI технологий. Простые, эффективные анимации. Идеальный баланс между доступом к инструментам и comprehensive website experience.
- **Можно стырить**: Нет (custom)
- **Платформа**: Custom

### 8. Craft.do
- **URL**: https://craft.do
- **Что крутого**: Soft beige фоны, subtle blue градиенты, card-based layout с rounded corners и glass-morphism. Paper textures, cloud illustrations, high-quality фотографии. Notebook-like feel.
- **Можно стырить**: Нет (custom)
- **Платформа**: Custom

### 9. Daylight (godaylight.com)
- **URL**: https://godaylight.com
- **Что крутого**: Тёплый beige/cream base + vibrant orange (#F66F00). Тройная typeface иерархия (serif/sans/mono включая Aeonik Pro). GSAP + ScrollTrigger, mask text reveals, Mux video. Sticky sections.

### 10. Rise Plasma

## ПЛАТФОРМЫ ДЛЯ КЛОНИРОВАНИЯ (подробно)

### 1. Framer — Remix/Clone Templates
- **Как**: File > Copy Remix Link (или Cmd+K > "Remix"). Получаешь 1:1 редактируемую копию со всеми слоями, анимациями, CMS, interactions.
- **Цены**: Free $0 (framer.app subdomain, badge, 1K visitors), Basic $10/mo, Pro $30/mo, Scale $100/mo
- **Ограничения**: Free план — нет custom domain, badge "Made in Framer". Нельзя экспортировать чистый HTML/CSS.
- **Лучше для**: No-code, быстрый старт, кастомизация шаблонов

### 2. Webflow — Cloneable Projects
- **Как**: Webflow Showcase имеет тысячи cloneable проектов. Кнопка "Clone in Webflow" → полный дубликат. Также: flowbase.co/clone, showcased.webflow.io
- **Цены**: Starter $0 (2 проекта, 2 страницы), Basic ~$18/mo, CMS ~$29/mo, Business ~$49/mo
- **Ограничения**: Free — 2 static pages, no custom domain, no code export
- **Лучше для**: Сложные анимации, interactions, CMS
